Hard Skills

Git

Git é um sistema de controle de versão de arquivos. Através deles podemos desenvolver projetos na qual diversas pessoas podem contribuir simultaneamente.

Github

O GitHub é usado principalmente para armazenar repositórios, que são as "localizações" dos arquivos de um determinado projeto. Os usuários podem criar projetos com base em outros já existentes e editá-los à sua maneira.

HTML

A Linguagem de marcação de Hipertexto (HTML) é uma linguagem que compõe a maior parte das páginas da internet e dos aplicativos online.

CSS

Cascading Style Sheets( ou simplesmente CSS), pela qual se define estilos para um projeto web (página de internet).

JAVASCRIPT

O JavaScript é uma linguagem de programação de alto nível voltada para o desenvolvimento web.

NODE.JS

É uma plataforma em que é possível criar aplicações Javascript sem depender de um browser para a execução. Sua execução em single-thread não exige resposta a cada requisição.

ALGORITMO

Um algoritmo é uma sequência de instruções bem definidas, normalmente usadas para resolver problemas de matemática específicos, executar tarefas, ou para realizar cálculos e equações.

JAVA

Como linguagem de programação, o código Java é baseado em classes e orientado a objetos, com foco em segurança, portabilidade e alta performance. Extensa biblioteca de rotinas e APIs para trabalhar com recursos de rede.
